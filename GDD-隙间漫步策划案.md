**据点房子类型一览：**

| 🏷️ 子类型 | ⚔️ 敌人配置 | 🎁 奖励 | 🗺️ 地图标识 | 占比 |
|:---|:---|:---|:---|:---|
| **标准据点** | 2-4 孢子兵 + 1 蘑菇骑士 | 标准命运之球（白/蓝） | `◼` 普通图标 | 25% |
| **强敌房** | 1 个巨型蘑菇骑士（类Boss） | 高级命运之球（蓝/紫/金） | `🍄` 巨型蘑菇图标 | 5% |
| **宝库房** | 无（解谜/钥匙） | 高级奖励 | `📦` 宝箱图标 | 3% |
| **事件房** | 无 | 随机事件结果 | `❓` 问号图标 | 2% |

> **💡 设计意图**：通路房提供喘息和资源收集空间；据点房是核心战斗节点，充当"关卡门禁"的角色；强敌房和宝库房是高风险高回报的可选目标。各类房间的配比确保探索节奏有张有弛。

#### 2.2.2 缩圈规则

| 规则项 | 详细说明 |
|:---|:---|
| ⏰ **触发时机** | 每天游戏时间到达「下午」节点（约开局后 2-3 分钟） |
| 🎯 **收缩方式** | 安全区从地图边缘向中心逐圈收缩（大逃杀式） |
| 🏁 **收缩终点** | 收缩至 **1 个房间大小**（必选通路房，排除据点房） |
| 💔 **圈外伤害** | 每秒受到 **最大 HP 的 5%～10%** |
| ⏱️ **缩圈速度** | 约 **60～90 秒** 完成完整收缩 |
| 🎨 **视觉表现** | 紫色孢子雾从边缘涌入，安全区边界有淡紫色光膜 |

#### 2.3.1 玩家操作映射

```
┌──────────────────────────────────────────────────┐
│                 🎮 操作映射表                      │

├────────────┼───────────────┼─────────────────────┤
│ 🏃 移动    │ WASD          │ 左摇杆              │

│ 🔥 法术2   │ E 键           │ Y 键                │
│ 🧪 消耗品  │ F 键           │ B 键                │
│ ✋ 交互    │ E 键（长按）   │ X 键（长按）        │

└──────────────────────────────────────────────────┘
```

> 💡 **设计要点**：攻击方向由鼠标/右摇杆朝向决定，近战攻击命中前方扇形范围。格挡消耗耐力，完美格挡（0.2s窗口）可反弹弹幕并恢复耐力。法术有独立冷却（CD），不受耐力限制。

## 🛠️ 五、技术架构

### 5.1 Godot 4.7 技术选型

| 需求领域 | 方案选择 | 说明 |
|:---|:---|:---|
| 🖼️ **2D 渲染** | `Node2D` + `TileMapLayer` | Godot 4.3+ 新版 TileMap 系统 |
| 🧱 **物理碰撞** | `CharacterBody2D` / `StaticBody2D` | 玩家用 Character，场景用 Static |
| 💥 **战斗系统** | `Area2D` + 碰撞检测 + ObjectPool 模式 | 近战判定用 Area 扇形，法术弹幕用对象池回收 |
| 🧠 **敌人AI** | `NavigationAgent2D` + 行为树 | 追踪/攻击/施法行为分离 |
| 🛡️ **格挡系统** | `Area2D` 碰撞 + 帧窗口判定 | 完美格挡用 0.2s 时间窗口 + `Engine.get_physics_frames()` |
| 🌐 **网络** | `ENetMultiplayerPeer` + RPC | Godot 内置网络方案 |
| 🖥️ **UI** | `Control` 节点 + Godot Theme | CanvasLayer 渲染 HUD |
| 💾 **存档** | `ConfigFile` / JSON | 轻量持久化 |
| 🔊 **音效** | `AudioStreamPlayer2D` + AudioBus | 三层 Bus：SFX / BGM / Ambient |

### 5.2 代码规范

```
✅ 使用 class_name 定义所有基类，方便全局引用
✅ 信号（Signal）解耦系统间通信（如：boss_died, circle_started）
✅ 资源文件（.tres）管理武器/敌人/房间配置数据
✅ @export 暴露可调参数，策划无需改代码即可调数值
✅ 每个 .gd 文件 ≤ 300 行，超过则拆分
```

### 挺进地牢（Enter the Gungeon）—— 地图结构参考

| 参考点 | 说明 |
|:---|:---|
| 🗺️ 房间制地图 | 房间通过门/通道连接，形成网状探索结构 |

| 🎁 房间清除奖励 | 完成房间挑战后掉落武器/被动/消耗品 |
| 💨 翻滚闪避 | 短暂无敌帧的闪避机制，保留为通用生存手段 |
| ⚠️ _不采用枪械战斗_ | _战斗核心改为 CRPG 近战 + 魔法_ |

### 黑夜君临（Elden Ring Nightreign）—— 节奏与联机参考

| 参考点 | 说明 |
|:---|:---|
| ⭕ 缩圈驱动 | 大逃杀式缩圈制造紧迫感，驱动玩家向心移动 |
